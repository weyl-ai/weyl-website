# Options

Module configuration options for weyl-std.

## weyl-std.formatter

```nix
weyl-std.formatter = {
  enable = true;         # Enable formatter module
  line-length = 100;     # Max line length
};
```

```nix

  cuda.capabilities = [ "8.9" "9.0" ];   # Target GPU capabilities
};
```
