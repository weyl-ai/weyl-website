Complete reference for API response formats.

## Sync Tier Response#

## Success (200)**Headers:**

```httpHTTP/1.1 200 OKContent-Type: image/webp  (or video/mp4)Content-Location: https://cdn.render.weyl.ai/i/abc123.webpContent-Length: 1234567X-Weyl-Request-Id: req_xyz789X-Weyl-Seed: 42X-Weyl-Duration-Ms: 847X-RateLimit-Remaining: 9950X-RateLimit-Reset: 1699564800

```**Body:*

* Binary image or video bytes#

## Capacity Exhausted (503)

```httpHTTP/1.1 503 Service UnavailableRetry-After: 30Link: <https://async.render.weyl.ai>; rel="alternate"Content-Type: application/json{  "error": "capacity_exhausted",  "message": "sync tier at capacity, retry in 30s or use async",  "request_id": "req_xyz789",  "retry_after": 30,  "alternate_tier": "https://async.render.weyl.ai"}

```

## Async Tier Responses#

## Queue Submission (202)

```httpHTTP/1.1 202 AcceptedLocation: https://async.render.weyl.ai/jobs/j_abc123Link: </jobs/j_abc123/events>; rel="monitor"Content-Type: application/json{  "id": "j_abc123",  "status": "queued",  "position": 5,  "eta_seconds": 75,  "events_url": "https://async.render.weyl.ai/jobs/j_abc123/events"}

```#

## Job Status (Queued)

```json{  "id": "j_abc123",  "status": "queued",  "position": 3,  "eta_seconds": 45,  "created_at": "2024-12-01T17:00:00Z"}

```#

## Job Status (Running)

```json{  "id": "j_abc123",  "status": "running",  "progress": 0.65,  "eta_seconds": 8,  "started_at": "2024-12-01T17:00:32Z",  "created_at": "2024-12-01T17:00:00Z"}

```#

## Job Status (Complete 

- 303 Redirect)

```httpHTTP/1.1 303 See OtherLocation: https://cdn.render.weyl.ai/i/abc123.webpX-Weyl-Seed: 42X-Weyl-Duration-Ms: 12400

```Follow `Location` header to retrieve output.#

## Job Status (Error)

```json{  "id": "j_abc123",  "status": "error",  "error": {    "code": "generation_failed",    "message": "CUDA out of memory",    "retriable": true  },  "created_at": "2024-12-01T17:00:00Z",  "started_at": "2024-12-01T17:00:32Z",  "completed_at": "2024-12-01T17:00:45Z",  "request": {    "prompt": "...",    "image": "..."  }}

```#

## Queue Full (429)

```json{  "error": "queue_full",  "message": "queue depth exceeded, retry in 60s",  "request_id": "req_xyz789",  "retry_after": 60}

```

## Content-Type Headers

| Output     | Content-Type ||------------|--------------|| Image      | `image/webp` || Video      | `video/mp4`  || JSON       | `application/json` || SSE Stream | `text/event-stream` |

## Rate Limit Headers

```X-RateLimit-Remaining: 9950      # requests remainingX-RateLimit-Reset: 1699564800    # unix timestampX-RateLimit-Limit: 10000         # total limit

```When rate limited (429):

```httpHTTP/1.1 429 Too Many RequestsRetry-After: 60X-RateLimit-Remaining: 0X-RateLimit-Reset: 1699564860{  "error": "rate_limited",  "message": "rate limit exceeded, retry in 60s",  "retry_after": 60}

```
