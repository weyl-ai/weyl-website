Reference for common types, enums, and identifiers.

## Modality

```typescripttype Modality = 'image' | 'video';

```Output type:

- `image` 

- Static image (JPEG, WebP)

- `video` 

- Video clip (MP4)

## Task

```typescripttype Task = 't2i' 

| 'i2i' | 'edit' | 't2v' | 'i2v';

```

| Task   | Requires            | Produces | Description                ||--------|---------------------|----------|----------------------------|| `t2i`  | prompt              | image    | text to image              || `i2i`  | prompt + image      | image    | image to image             || `edit` | prompt + image + mask | image  | inpaint / outpaint         || `t2v`  | prompt              | video    | text to video              || `i2v`  | prompt + image      | video    | image to video             |

## Identifiers#

## Request ID

```Format: req_[a-zA-Z0-9]{12}Example: req_xyz789abc123

```Generated by server, included in `X-Weyl-Request-Id` header.#

## Job ID

```Format: j_[a-zA-Z0-9]{12}Example: j_7Xy9kL2mN4pQ

```**Async tier only.*

* Generated by server when job queued.#

## User ID

```Format: u_[a-zA-Z0-9]{12}Example: u_abc123xyz789

```Associated with API key.#

## Asset ID

```Format: a_[a-zA-Z0-9]{12}Example: a_def456uvw012

```For uploaded assets.

```**Examples:**

```https://cdn.render.weyl.ai/i/abc123xyz789.webphttps://cdn.render.weyl.ai/v/def456uvw012.mp4https://cdn.render.weyl.ai/a/ghi789rst345

## Seed

## Progress

```typescripttype Progress = number;  // 0.0 to 1.0

```#

## Guidance

```Model-specific ranges:

- FLUX Dev/Dev2: 1.0 

- 5.0

- FLUX Schnell: 3.5 (fixed)

- WAN: 6.0 

- 9.0

- Z-Image: 1.0 (fixed)#

```#

## Steps

```typescripttype Steps = number;  // Model-specific range

```

| Model        | Min | Max | Default ||--------------|-----|-----|---------|| flux/schnell | 4   | 4   | 4       || flux/dev     | 10  | 50  | 25      || flux/dev2    | 10  | 50  | 25      || zimage/turbo | 4   | 15  | 8       || wan/default  | 10  | 50  | 20      |#

## Duration (Video)

```typescripttype Duration = number;  // 0.5 to 10.0 seconds

```#

## Strength (i2i)

```typescripttype Strength = number;  // 0.0 to 1.0

```

| Range   | Effect                     ||---------|----------------------------|| 0.1-0.3 | Subtle changes             || 0.4-0.6 | Balanced transformation    || 0.7-0.9 | Heavy transformation       || 1.0     | Maximum (nearly ignores source) |#

## Complete Request Type

```typescriptinterface GenerateRequest {  // Required  prompt: string;    // Task-specific  image?: string;               // URL or data URI  mask?: string;                // URL or data URI    // Generation params  negative_prompt?: string;  steps?: Steps;  guidance?: Guidance;  cfg?: CFG;                    // alias for guidance  seed?: Seed;  strength?: Strength;          // i2i only  duration?: Duration;          // video only    // Advanced  sampler?: Sampler;  scheduler?: Scheduler;  lora?: LoRAConfig[];  detail_amount?: number;       // 0.0-1.0  detail_start?: number;        // 0.0-1.0  detail_end?: number;          // 0.0-1.0    // Multi-output  count?: number;               // 1-4    // Video-specific  motion_strength?: number;     // 0.0-1.0  lightning?: boolean;          // WAN only  shift?: number;               // 0-10, bong_tangent only}

```

## Complete Job Type

```typescriptinterface Job {  id: JobId;  status: JobStatus;    // Queue info  position?: number;  eta_seconds?: number;    // Progress  progress?: Progress;    // Output  output?: CDNUrl;  outputs?: CDNUrl[];  seed?: Seed;    // Error  error?: {    code: string;    message: string;    retriable: boolean;  };    // Timestamps  created_at: ISO8601;  started_at?: ISO8601;  completed_at?: ISO8601;    // Debug  request?: GenerateRequest;}

```
