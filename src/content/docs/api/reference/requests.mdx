Complete reference for request body schemas across all endpoints.

## Text-to-Image (t2i)

```typescriptinterface T2IRequest {  prompt: string;                // 1-4000 characters  negative_prompt?: string;      // max 2000 characters  steps?: number;                // model-specific range  guidance?: number;             // 1.0-5.0 typically  cfg?: number;                  // alias for guidance  seed?: number;                 // 0-4294967295  sampler?: 'euler' 

| 'res_2m' | 'res_3s';  scheduler?: 'simple' | 'beta' | 'exponential' | 'sgm_uniform';  count?: number;                // 1-4, number of images  lora?: LoRAConfig[];  detail_amount?: number;        // 0.0-1.0  detail_start?: number;         // 0.0-1.0  detail_end?: number;           // 0.0-1.0}

```**Example:**

```json{  "prompt": "portrait of a woman, natural lighting",  "negative_prompt": "blur, distortion, artifacts",  "steps": 25,  "guidance": 3.5,  "seed": 42,  "sampler": "res_2m",  "scheduler": "exponential"}

```

## Image-to-Image (i2i)

```typescriptinterface I2IRequest extends T2IRequest {  image: string;                 // URL or data URI  strength?: number;             // 0.0-1.0, default 0.7}

```**Example:**

```json{  "prompt": "convert to watercolor style",  "image": "https://example.com/photo.jpg",  "strength": 0.7,  "guidance": 3.5}

```

## Inpaint / Edit

```typescriptinterface EditRequest extends T2IRequest {  image: string;                 // source image  mask: string;                  // mask image (white = edit, black = keep)}

```**Example:**

```json{  "prompt": "add neon sign saying OPEN",  "image": "https://example.com/storefront.jpg",  "mask": "https://example.com/mask.png"}

```

## Image-to-Video (i2v)

```typescriptinterface I2VRequest {  prompt: string;                // motion description  image: string;                 // source image  duration?: number;             // 0.5-10.0 seconds  steps?: number;  cfg?: number;                  // 6.0-8.0 for WAN  sampler?: 'uni_pc' 

| 'euler' | 'res_2m' | 'res_3s';  scheduler?: 'simple' | 'beta' | 'bong_tangent';  motion_strength?: number;      // 0.0-1.0  seed?: number;  lightning?: boolean;           // enable lightning mode  shift?: number;                // 0-10, for bong_tangent}

```**Example:**

```json{  "prompt": "she slowly turns to face the camera",  "image": "https://example.com/portrait.jpg",  "duration": 3,  "cfg": 7.0,  "sampler": "res_2m",  "scheduler": "bong_tangent"}

```

## Text-to-Video (t2v)

```typescriptinterface T2VRequest {  prompt: string;  duration?: number;  steps?: number;  cfg?: number;  sampler?: string;  scheduler?: string;  seed?: number;}

```**Currently not supported*

* - WAN is i2v only.

## LoRA Config

```typescriptinterface LoRAConfig {  id: string;                    // LoRA identifier  weight: number;                // 0.0-1.0+, typically 0.7-0.9}

```**Example:**

```json{  "lora": [    {"id": "anti_blur", "weight": 0.8},    {"id": "realism_boost", "weight": 0.75}  ]}

```

## Async Queue Request

```typescriptinterface AsyncQueueRequest {  modality: 'image' 

| 'video';  family: string;  model: string;  task: string;  format?: string;  prompt: string;  image?: string;  mask?: string;  [key: string]: unknown;        // generation params    // Queue-specific  idempotency_key?: string;  webhook?: string;  priority?: 'low' | 'normal' | 'high';  ttl?: number;                  // seconds}

```**Example:**

```json{  "modality": "video",  "family": "wan",  "model": "default",  "task": "i2v",  "format": "720p",  "prompt": "she turns to face camera",  "image": "https://example.com/portrait.jpg",  "cfg": 7.0,  "webhook": "https://myapp.com/callback",  "priority": "normal",  "idempotency_key": "batch_1_item_5"}

```

## Validation Rules#

## Prompt

- Min length: 1 character

- Max length: 4000 characters

- Must be valid UTF-8#

## Negative Prompt

- Max length: 2000 characters

- Optional#

## Image URLs

- Must be HTTPS (or data URI)

- Max file size: 10 MB

- Supported formats: JPEG, PNG, WebP#

## Mask Images

- Must match source image dimensions

- White pixels = edit region

- Black pixels = preserve region

- Format: PNG or WebP#

## Steps

| Model        | Min | Max  | Default ||--------------|-----|------|---------|| flux/schnell | 4   | 4    | 4       || flux/dev     | 10  | 50   | 25      || flux/dev2    | 10  | 50   | 25      || zimage/turbo | 4   | 15   | 8       || wan/default  | 10  | 50   | 20      |#

## Guidance / CFG

| Model        | Min  | Max  | Default ||--------------|------|------|---------|| flux/schnell | 3.5  | 3.5  | 3.5     || flux/dev     | 1.0  | 5.0  | 3.5     || flux/dev2    | 1.0  | 5.0  | 3.5     || zimage/turbo | 1.0  | 1.0  | 1.0     || wan/default  | 6.0  | 9.0  | 7.0     |#

## Duration (Video)

- Min: 0.5 seconds

- Max: 10.0 seconds

- Default: 2.0 seconds#

## Seed

- Min: 0

- Max: 4294967295 (2^32 

- 1)

- Optional 

- random if not specified#

## Count (Multi-image)

- Min: 1

- Max: 4

- Default: 1

## Common Errors#

## 400 Bad Request

```json{  "error": "validation_failed",  "message": "Invalid request format",  "details": {    "field": "prompt",    "reason": "prompt exceeds max length of 4000 characters"  }}

```**Common validation errors:**

- Prompt too long

- Invalid image URL

- Steps out of range

- Guidance out of range

- Invalid sampler/scheduler

- Missing required field#

## 413 Payload Too Large

```json{  "error": "payload_too_large",  "message": "Request body exceeds 10MB limit"}

```Use `/uploads` endpoint for large images.
