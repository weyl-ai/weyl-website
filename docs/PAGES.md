# Pages Documentation

This document describes all routes and pages in the Weyl AI website.

## Marketing Pages

### Landing Page `/`
**File**: `src/pages/index.astro`

The main landing page featuring:
- Hero section with value proposition
- Trust signals and metrics (TrustCascade)
- Use cases (DemandCurves)
- Technology stack overview
- Call-to-action sections

**Theme**: Hypermodern (dark mode with Iosevka monospace)

### Brand Assets `/brand`
**File**: `src/pages/brand.astro`

Brand asset download page providing:
- Logo files in various formats
- Monotile (icon) assets
- Wordmark assets
- Brand guidelines and usage instructions
- Color specifications

### Posters `/posters`
**File**: `src/pages/posters.astro`

Downloadable poster gallery:
- Technical architecture posters
- Educational materials
- Promotional graphics

### Archive `/archive`
**File**: `src/pages/archive.astro`

Special showcase page:
- **Theme**: High Modernism (light mode)
- Demonstrates alternate design system
- System font stack (Helvetica/Arial)
- Historical/archival content presentation

### Request Access `/request-access`
**File**: `src/pages/request-access.astro`

API access request form:
- User information collection
- Use case description
- Integrates with Klaviyo or Slack Connect
- Submits to `/api/request-access` endpoint

See: `API_REQUEST_ACCESS_SETUP.md` and `SLACK_CONNECT_SETUP.md` for integration setup

### Sitemap `/sitemap`
**File**: `src/pages/sitemap.astro`

Human-readable sitemap page:
- Lists all major routes
- Organized by category
- Provides quick navigation
- Complements XML sitemaps

## Blog

### Blog Index `/blog`
**File**: `src/pages/blog/index.astro`

Blog post listing page:
- Lists all published blog posts
- Displays post metadata (date, author, excerpt)
- Supports pagination (if configured)
- Links to individual posts

### Blog Posts `/blog/[slug]`
**File**: `src/pages/blog/[...slug].astro`

Individual blog post pages:
- Dynamic route catches all blog slugs
- Renders MDX content from `src/content/blog/`
- Includes reading time estimate
- Last updated timestamp
- Syntax-highlighted code blocks
- Social sharing metadata

**Example posts**:
- `/blog/introducing-weyl-ai` - Launch announcement

### RSS Feed `/blog/rss.xml`
**File**: `src/pages/blog/rss.xml.ts`

XML RSS feed for blog:
- Lists all blog posts with metadata
- Full content or excerpts
- Enables RSS reader subscriptions

**File**: `src/pages/ai-sitemap.xml.ts`

## Page Organization

### By Theme
- **Hypermodern (Dark)**: `/`, `/blog`, `/docs`, most pages
- **High Modernism (Light)**: `/archive`

### By Purpose
- **Marketing**: `/`, `/brand`, `/posters`
- **Content**: `/blog/*`, `/docs/*`
- **Utility**: `/sitemap`, `/request-access`
- **SEO/AI**: `/llms.txt`, `/ai-sitemap.xml`, `/.well-known/*`

### By Rendering
- **Static (prerendered)**: All content pages, marketing pages
- **Server-rendered**: `/api/*` endpoints
- **Dynamic**: `/blog/[slug]`, `/api/og/[slug]`

## Performance Considerations

### Static Generation
- Most pages are pre-rendered at build time
- Fast initial load, excellent SEO
- No server round-trips for content

### View Transitions
- Smooth SPA-like navigation
- Enabled via Astro ViewTransitions
- Preserves scroll position
- Animates page changes

### Image Optimization
- All images optimized via Sharp
- Lazy loading enabled
- Responsive images with srcset
- Modern formats (WebP) where supported

### Font Loading
- Custom Iosevka fonts preloaded
- Fallback system fonts
- Font-display: swap for fast rendering
- WOFF2 format for best compression
